## 📌 基础爽点模式

### 1. 打脸反转 (Face Slap)

```markdown
【模板】
主角原本被嘲笑/轻视/欺负→展示实力→对方震惊/后悔

【经典场景】
- 宴会上被当成穷酸女婿→亮出身份/实力
- 比赛中被对手嘲讽→关键时刻逆转
- 职场被同事排挤→展示能力惊艳全场

【情绪曲线】
轻视(铺垫) → 冲突(积累) → 爆发(高潮) → 爽感(满足)
```

### 2. 英雄救美 (Hero Saving Beauty)

```markdown
【模板】
美女遇难 → 主角出手 → 美女倾心 → 敌人后悔

【关键要素】
- 危机要足够危险（否则体现不出价值）
- 出手要足够帅气（吸引眼球）
- 美女反应要到位（仰慕/心动）
- 围观者要震惊（增加面子）
```

### 3. 获得奇遇 (Fortune Acquisition)

```markdown
【模板】
误入秘境/触发机关/偶遇高人 → 获得传承/宝藏/功法

【类型】
- 传承类：戒指老爷爷、秘境传承、上古血脉
- 资源类：天材地宝、神兵利器、稀有功法
- 能力类：特殊体质、超级系统、无敌金手指
```

### 4. 越级挑战 (Cross-Level Battle)

```markdown
【模板】
低境界vs高境界 → 众人不看好 → 逆袭反杀

【爽点来源】
- 以弱胜强的戏剧性
- 围观者的震惊
- 敌人的不可置信
- 主角的淡定点评
```

### 5. 身份打脸 (Identity Reveal)

```markdown
【模板】
隐藏身份被质疑 → 亮明身份 → 震惊全场

【常用身份】
- 豪门继承者/隐世家族
- 顶级强者/传说人物
- 神秘组织高层
- 首富/幕后boss
```

## 🔥 复合爽点组合

### 组合A: 退婚流

```
【经典模板】
1. 被未婚妻/家族退婚（耻辱）
2. 展示潜力/实力（反转）
3. 对方后悔/挽回（爽点）
4. 主角拒绝/嘲讽（高潮）

【情绪起伏】
屈辱 → 不甘 → 爆发 → 满足
```

### 组合B: 系统流

```
【模板】
1. 系统激活/任务发布
2. 完成任务的奖励
3. 实力暴涨/商城购物
4. 装逼打脸

【系统奖励类型】
- 积分/金币（购买力）
- 抽奖（惊喜感）
- 碎片/成就（收集欲）
- 技能/血脉（成长感）
```

### 组合C: 修仙升级流

```
【模板】
1. 修炼瓶颈 → 奇遇突破
2. 境界提升 → 功法领悟
3. 天材地宝 → 实力暴涨
4. 越级战斗 → 震惊众人

【爽点节奏】
小瓶颈(5%) → 小突破(15%) → 
大瓶颈(40%) → 大奇遇(50%) → 
升级(70%) → 战斗检验(85%) → 震惊(100%)
```

## ✍️ 爽点写作技巧

### 1. 情绪铺垫公式

```
正常(30%) → 被轻视(40%) → 冲突(60%) → 
反转(80%) → 爽点(100%)
```

### 2. 描写强度递增

```
旁观者视角：看热闹 → 发现异常 → 震惊
主角视角：蓄力 → 爆发 → 收尾
```

### 3. 爽点密度

> 每1000字至少2个爽点

| 章节位置 | 爽点密度 |
|----------|----------|
| 开头 | 1个（设悬念） |
| 中段 | 2-3个（维持节奏） |
| 结尾 | 1个（留悬念） |

## 🎯 爽点触发器库

### 语言类

- "这...这不可能！"
- "你到底是什么人？"
- "竟然是他！"
- "这、区区蝼蚁..."
- "不可能，绝对不可能！"

### 动作类

- 脸色大变 / 瞳孔收缩
- 倒吸一口凉气
- 后退三步
- 手中的东西掉落

### 心理类

- 后悔莫及
- 如坠冰窟
- 悔不当初
- 震惊欲绝

## 🚫 避开毒点

### 1. 圣母主角

❌ 原谅所有敌人
✅ 有仇必报，快意恩仇

### 2. 水字数

❌ 无关细节写太多
✅ 紧扣主线，节奏紧凑

### 3. 套路老化

❌ 退婚，打脸，再退婚
✅ 创新组合，反转再反转

### 4. 实力崩坏

❌ 前期无敌，后期也无敌
✅ 稳步升级，合理越级
